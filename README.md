# Java Generics Type Safety Demo

This example demonstrates the core benefits of Java Generics. It compares the unsafe pre-Java 5 approach (using raw Object and manual casting) with modern, type-safe generic classes, bounded type parameters, and generic methods.

## Language

`java`

## How to Run

Compile the code using 'javac GenericsDemo.java' and run it using 'java GenericsDemo'
